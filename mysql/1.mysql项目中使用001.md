#### 3.去重数据

* 只保存一条航线

  |      | SELECT                          |
  | ---- | ------------------------------- |
  |      | *                               |
  |      | FROM                            |
  |      | ydyl_planlines_copy a           |
  |      | WHERE                           |
  |      | EXISTS (                        |
  |      | SELECT                          |
  |      | 1                               |
  |      | FROM                            |
  |      | ydyl_planlines_copy b           |
  |      | WHERE                           |
  |      |                                 |
  |      | b.start_code = a.end_code       |
  |      | AND b.end_code = a.start_code   |
  |      | AND a.ID > b.id                 |
  |      | )                               |
  |      | OR                              |
  |      | NOT EXISTS (                    |
  |      | SELECT                          |
  |      | 1                               |
  |      | FROM                            |
  |      | ydyl_planlines_copy b           |
  |      | WHERE                           |
  |      |                                 |
  |      | b.start_code = a.end_code       |
  |      | AND b.end_code = a.start_code   |
  |      | )                               |
  |      |                                 |
  |      | GROUP BY id,start_code,end_code |
  |      | ORDER BY id                     |
